# Correctly - Grammar Checking Chrome Extension

## Users
- Writers, students, professionals who want to improve their writing
- Non-native English speakers seeking grammar assistance
- Anyone who writes text in web applications and wants real-time feedback

## Product Purpose
A Chrome extension that provides real-time grammar checking and suggestions as users type in web applications, helping them write with greater accuracy and confidence.

## Tone
- Helpful and educational
- Non-intrusive and supportive
- Professional yet approachable

## Anti-References
- Overly aggressive grammar correction that interrupts flow
- Complex interfaces that distract from writing
- Extensions that collect excessive user data

## Strategic Principles
- Focus on core grammar checking excellence
- Maintain lightweight performance
- Respect user privacy and data
- Provide clear, actionable suggestions
- Integrate seamlessly with writing workflows
